# Create New Skill

Create a new reusable skill/capability.

## Instructions

1. Analyze the description to understand the desired capability.
2. Scaffold the new skill with appropriate structure:
   - `SKILL.md` - Main skill documentation
   - `scripts/` - Helper scripts (if needed)
   - `references/` - Reference documentation (if needed)
3. Implement the skill logic.
4. Validate the skill works as expected.
